# big event

```   
项目要实现的目标:完成登录注册事件,利用ajax获取信息,个人中心模块
			  可以实现获取信息并编辑,修改头像和重置密码;文章				模块可以添加和修改文章分类.发布文章,修改或者删			  除自己的文章.
主要使用的技术:AJAX 
使用到的库:JQuery,layui,art-template,JQ的cropper插件,tinymce,nprogess等
遇到的困难:修改文章时出现了无法加载内容的问题,发现多了一个div的单独标签,修复成功后发现无法修改内容,查证后发现把富文本初始化放在函数外边可以解决.发现图片裁剪后再次修改文章发现图片没有裁剪,后来发现是路径问题,不应该把图片的src写死.应该把src属性的值动态获取.
```
